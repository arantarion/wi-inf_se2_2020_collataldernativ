<h2>Nutzung</h2>

<h4>1.    Einleitung - Was ist Coll@Aldernativ?</h4>

Coll@Aldernativ bietet Studenten und Unternehmen eine zentrale Schnittstelle zur Jobvermittlung bzw. Mitarbeitersuche.
Nach unserem Motto "Hier findet jeder seinen Traumjob" haben wir als Projektteam uns stehts bemüht mit unseren Werten und
Vorstellungen jedem die Möglichkeit zu bieten mehr aus seinem Arbeitgeber bzw. Arbeitnehmer rauszuholen und diese perfekt 
aufeinander abzustimmen.

<h5>1.1   Homepage</h5>

Sollten Sie uns besuchen finden Sie auf unsere Homepage direkt zentral gelegen ein Login-Fenster zur Eingabe Ihrer persönlichen
Daten. Zur Erstellung eines neuen Accounts auf unsere Website benutzen Sie bitte den Button "Registrieren" und fahren Sie
mit Schritt [2] fort. Sollten sie bereits einen Account haben so führen sie mit Punkt [3] fort.

<h4>2.   Registrierung</h4>

Nachdem Sie auf der Homapage den Button 'Registrieren' angeklickt haben, befinden Sie sich nun in einer Auswahl.
Hier entscheiden Sie, ob sie als 'Student' später nach Jobangeboten suchen möchten, oder als 'Unternehmer' selbst Jobs 
anzubieten haben. Dies ist eine endgültige Entscheidung und kann nicht Rückgängig gemacht oder zu einem späteren Zeitpunkt
verändert werden!
Klicken Sie auf die Rolle, für welche Sie sich registrieren möchten und fahren sie dementsprechend mit Schritt [2.1] bzw [2.2]
fort.

<h5>2.1   .. als Student</h5>

Wenn Sie sich dazu entschieden haben sich als 'Student' zu registrieren, befinden Sie sich nun im zweiten Schritt der Registrierung.
Hier geben sie ihren Nutzernamen auf der Plattform, ihre E-Mail-Adresse und ein persönliches Passwort ein.
Dieses Passwort benötigt mindestens einen Groß- bzw Kleinbuchstaben und eine Zahl und muss insgesamt mind. 8 Zeichen umfassen,
damit unsere Sicherheitsstandard und die Ihrer persönlichen Daten gewährleistet werden können.

Anschließen klicken Sie auf 'weiter'.

Im nächsten Schritt geben Sie Ihre persönlichen Daten ein um möglichen Arbeitgebern Ihre Daten zu übermitteln.
Hierunter fällt Ihr Name, Geburtstag und Anschrift.

Nach einem weiteren Klick auf 'weiter' gelangen Sie zur Homepage zurück und bekommen eine Meldung, dass sie sich erfolgreich
registriert haben.
Willkommen auf <b>Coll@Aldernativ</b>!

Für einen erfolgreichen Login, lesen Sie weiter unter Schritt [3]

<h5>2.2   .. als Unternehmen</h5>

Wenn Sie sich dazu entschieden haben sich als 'Student' zu registrieren, befinden Sie sich nun im zweiten Schritt der Registrierung.
Hier geben sie ihren Nutzernamen auf der Plattform, Ihre E-Mail-Adresse und ein persönliches Passwort ein.
Dieses Passwort benötigt mindestens einen Groß- bzw Kleinbuchstaben und eine Zahl und muss insgesamt mind. 8 Zeichen umfassen,
damit unsere Sicherheitsstandard und die Ihrer persönlichen Daten gewährleistet werden können.

Anschließen klicken Sie auf 'weiter'.

Im nächsten Schritt geben Sie den Namen, die Anschrift, sowie eine kurze Beschreibung ihres Unternehmen eins.
Werden Sie kreativ um möglichst viele Bewerber aufmerksam auf Sie zu machen!
Optional können sie auch ihre Unternehmensseite verlinken. 

Nachdem Sie erneut ihre Angaben überprüft haben und auf 'weiter' geklickt haben, bekommen Sie eine Meldung, dass Sie sich erfolgreich
registriert haben.
Willkommen auf <b>Coll@Aldernativ</b>!

Für einen erfolgreichen Login, lesen Sie weiter unter Schritt [3]

<h4>3.    Login</h4>

Auf unserer Homepage befindet sich in der Mitte ein Login-Fenster. Hier müssen die lediglich Ihre, bei der Registrierung, 
angegebenen Daten eingeben für einen erfolgreichen Login auf Coll@Aldernativ!
Anschließend werden Sie automatisch auf unsere MainPage [4] weitergeleitet

<h4>4.    MainPage</h4>

Nach einem erfolgreichen Login aus Schritt [3] werden sie auf unsere MainPage weitergeleitet.
Für die Erklärung schlagen sie als Student [4.1] bzw. als Unternehmer [4.2] auf.

<h5>4.1   .. als Student</h5>

Als Student haben sie die Möglichkeit sich auf eine Stellenanzeige zu bewerben über die Suchfunktion.
Für eine Erklärung der Suchfunktion [4.1.1] nachschlagen, für eine direkte Bewerbung [4.2.2]
Sollten Sie neu auf unserer Plattform sein und noch keine Details über Ihren Studiengang bzw. wichtige Dokumente (z.B.
Lebenslauf) hochgeladen haben so empfehlen wir Ihnen zuerst Ihr Profil unter dem Schritt [5.1] zu bearbeiten.

<i>4.1.1 Suchfunktion benutzen</i>

Geben Sie einen Suchbegriff ein um in unserem System nach Stellenanzeigen zu suchen, welche Ihr eingegebenes Stichwort
als Unternehmensname, Bereich, Kontakt, Beschreibung oder Name in der Stellenanzeige beinhaltet.
Sie können in der Tabelle die gefundenen Einträge nach jedem Kriterium auf- bzw. absteigend sortieren.

<i>4.1.2 Bewerbung auf eine gefundene Stellenanzeige</i>

Nachdem Sie eine passende Stellenanzeige gefunden haben klicken Sie die Stellenanzeige an, welche Sie anspricht.
Diese wird nun farbig unterlegt. Anschließend klicken Sie unten links neben der Tabelle auf 'Direkt zur Bewerbung'.
Es öffnet sich ein Bewerbungsfenster, in welchem Sie ein optionales Motivationschreiben ihrem vielleicht zukünftigen Arbeitgeber
zukommen lassen können!
Falls Sie sich es anders überlegt haben, können sie oben rechts das Fenster zur Bewerbung schließen, oder auf den Button 
'zurück zur Hauptseite' klicken.
Wenn Sie mit Ihrer Bewerbung zufrieden sind, dann müssen Sie diese lediglich abschicken, indem Sie auf den Button 
'Bewerbung senden' klicken. 

<h5>4.2   .. als Unternehmen</h5>

Als Unternehmer haben sie die Möglichkeit sich nach Stellenanzeigen umzuschauen bzw. passenden Studenten
für ihre Firma zu suchen, über die Suchfunktion.
Für eine Erklärung der Suchfunktion [4.2.1] nachschlagen.
Sollten Sie neu auf unserer Plattform sein und noch keine Details über Ihre Firma bekanntgegeben haben bzw. noch keine
Stellenanzeige erstellt haben, so empfehlen wir Ihnen zuerst Ihr Profil unter dem Schritt [5.2] zu bearbeiten.

<i>4.2.1 Suchfunktion benutzen</i>

Geben Sie einen Suchbegriff ein um in unserem System nach Stellenanzeigen zu suchen, welche ihren eingegebenes Stichwort
als Unternehmensname, Bereich, Kontakt, Beschreibung oder Name in der Stellenanzeige beinhaltet.
Sie können in der Tabelle die gefundenen Einträge nach jedem Kriterium auf- bzw. absteigend sortieren.

Zusätzlich befinden sich unter der Tabelle für Stellenanzeigen in Ihrer Sicht noch eine weitere Liste mit allen 
registrierten und aktiven Nutzern.
Sie können in der Tabelle die gefundenen Einträge nach jedem Kriterium auf- bzw. absteigend sortieren.

<h4>5.    Profilansicht</h4>

Durch einen Klick auf den Button „Mein Profil bearbeiten“ , oben auf der Seite, gelangt Sie auf Ihr Profil.

<h5>5.1.  .. als Student </h5>

Hier sehen Sie (wenn eingetragen) den Namen, das Studienfach, das Fachsemester, den aktuellen Job und den aktuellen Arbeitgeber.

<i>5.1.1 Profilangaben bearbeiten </i>

Durch einen Klick auf den Button zur Bearbeitung des Profils, (sieht aus wie ein kleines Buch mit einem Stift) öffnet sich 
ein Fenster in dem Sie die E-Mail-Adresse, das Studienfach, das Fachsemester und das Passwort ändern können.

<i>5.1.2 Upload von Dokumenten </i>

In oben genanntem Fenster ist es ebenfalls möglich, durch den Button „Hochladen“ ein Profilbild zum Profil hinzuzufügen, 
welches auch als Vorschau angezeigt wird. Ebenfalls ist es möglich, unter „Lebenslauf“ über den Button „Hochladen“ eine PDF-Datei hochzuladen, 
welche bei Bewerbungen mitgeschickt wird.

<i>5.1.3 Speicherung der geänderten Angaben </i>

Durch einen Klick auf den Button „Speichern“ werden alle bearbeiteten Angaben übernommen und im Profil angezeigt. 
Wenn Felder fehlerhaft ausgefüllt wurden, erscheint eine Fehlermeldung.

<h5>5.2   .. als Unternehmen </h5>

Hier sehen Sie den Namen des Unternehmens, die Website und eine Beschreibung des Unternehmens. 
Darunter befindet sich eine Tabelle mit allen eingestellten Stellenanzeigen.

<i>5.2.1 Profilangaben bearbeiten </i>

Durch einen Klick auf den Button zur Bearbeitung des Profils, (sieht aus wie ein kleines Buch mit einem Stift) öffnet sich ein Fenster 
indem Sie den Namen des Unternehmens, die Beschreibung, die E-Mail-Adresse, die Website und das Passwort ändern können.

<i>5.2.2 Upload von Dokumenten </i>

In oben genanntem Fenster ist es ebenfalls möglich, durch den Button „Foto hochladen“ ein Profilbild zum Profil hinzuzufügen.

<i>5.2.3 Speicherung der geänderten Angaben</i>

Durch einen Klick auf den Button „Speichern“ werden alle bearbeiteten Angaben übernommen und im Profil angezeigt. 
Wenn Felder fehlerhaft ausgefüllt wurden, erscheint eine Fehlermeldung.

<i>5.2.4 Erstellen von Stellenanzeigen</i>

Auf dem Profil ist es möglich durch den Button „Neue Stellenanzeige erstellen“ eine Stellenanzeige zu veröffentlichen. 
Sie gelangt auf eine neue Seite in der Sie Bereich, Kontakt, Beschreibung, Name, Gehalt und Start Datum der Stellenanzeige bestimmen können. 
Durch einen Klick auf „Speichern“ wird diese bestätigt und wird nun in der Tabelle auf dem Profil angezeigt.

<i>5.2.5 Löschen von Stellenanzeigen</i>

Auf dem Profil ist es in der Tabelle der Stellenanzeigen möglich, durch klicken auf die Kästchen neben den Anzeigen, diese zu markieren. 
Durch den Button „Ausgewählte löschen“ können Sie nun alle markierten Anzeigen löschen.

<h5>5.3   Löschung des Profils</h5>

Um ein Profil zu löschen, klicken Sie oben auf den Button „Mein Profil bearbeiten“ und gelangt in das eigene Profil. Von dort findet sich ein Button zur Bearbeitung des Profils, (sieht aus wie ein kleines Buch mit einem Stift) wodurch sich ein Fenster zur Bearbeitung öffnet. Durch einen Klick auf den Button „Profil löschen“ kommt eine Abfrage die mit „Ja“ bestätigt werden muss. Nach Eingabe des Passwortes und der Bestätigung ist nun das Profil gelöscht.

<h4>6.    Logout</h4>

Um sich nach der Nutzung der Plattform wieder auszuloggen, reicht ein Klick auf den Button „Logout“ oben auf der Seite. 
Im Anschluss gelangen Sie wieder auf die Login-Seite, von der es möglich ist, sich wieder erneut anzumelden oder zu registrieren.
